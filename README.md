# Daona — VN Gestionnaire de tâches

Application de gestion de tâches en ligne (Kanban), conçue avec le design Apple.

## Fonctionnalités

- 📋 **Tableau Kanban** — Colonnes personnalisables (Aujourd'hui, Demain, Terminé…)
- 📅 **Calendrier** — Vue mensuelle des tâches planifiées
- ⏱ **Chronomètre** — Suivi du temps par tâche avec camembert visuel
- 🔍 **Filtres & recherche** — Par colonne, par statut, par priorité
- ↻ **Récurrence** — Tâches quotidiennes, hebdomadaires, mensuelles
- ✅ **Sous-tâches** — Checklist avec progression
- 📤 **Export CSV** — Téléchargement des données
- 🗄 **Archive** — Historique des journées clôturées
- 🌙 **Mode sombre** — Adaptation automatique au thème système
- 📱 **Mobile** — Interface responsive
